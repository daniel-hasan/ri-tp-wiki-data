MoinMoin é um software para sistemas wiki. Com ferramentas avançadas de colaboração para editar web páginas como: revisor ortográfico, deleção de cache, impressão, anexos e outros. MoinMoin é software livre escrito em Python e licenciado sob a GNU GPL.

Há um plugin MediaWiki que torna o MoinMoin compatível com a linguagem de marcação do MediaWiki, isto é, o plugin consiste em um parser que habilita a interpretação de marcação do MediaWiki pelo MoinMoin, embora falte algum suporte (como suporte a InterWiki, por exemplo).[1] Assim, é possível usar dados do MediaWiki com o MoinMoin.[2]
