Allmusic (antiga All Music Guide) é uma grande base de dados em metadata sobre música, parte do grupo All Media Guide. Foi fundado por Michael Erlewine em 1991, como um guia para consumidores, e publicou seu primeiro livro de referência no ano seguinte. Considerada o padrão de facto para metadata musical, a base de dados da Allmusic é licenciada e usada em centenas de milhares de lojas de música e páginas de compra online.[1] A base de dados consiste em:

  • Metadata Básico: nomes, gêneros, créditos, informação de direitos autorais, número do produto, entre outros.
  • Conteúdo Descritivo: tons, humores, entre outros.
  • Conteúdo Relacionado: artistas e álbuns similares, influências, entre outros.
  • Conteúdo Editorial: biografias, resenhas, rankings, entre outros.

O Allmusic também possui o maior arquivo digital de música do mundo, com 5,5 milhões de canções completamente digitalizadas, e também a maior biblioteca de capas de álbuns com mais de meio milhão de capas.

A página Allmusic.com é uma amostra do que está disponível na base de dados. É usado extensivamente por gravadoras, DJs, detentores de direitos autorais, pesquisadores e outros membros da indústria musical, assim como fãs da música em geral. A página foi lançada em 1995 como uma demonstração online para licenciamentos em potencial da largura do conteúdo disponível na base de dados. Os dados do Allmusic também são usados pelos programas Windows Media Player e Musicmatch Jukebox para identificar e organizar coleções musicais.[1]
