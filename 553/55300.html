Em informática um(a) controlador(a) é um dispositivo de hardware que faz de interface entre o exterior de um dispositivo e o seu funcionamento interno. Por exemplo, para um microprocessador apenas enviar o endereço físico de dados para um disco rígido, o controlador (placa de circuitos anexa) traduz esse endereço e acciona os dispositivos mecânicos específicos do disco para que os dados possam ser enviados para o processador.
