AbiWord é um processador de texto de código­fonte aberto portanto, um software livre licenciado segundo a GPL. Roda em plataformas GNU/Linux, Mac OS, Microsoft Windows, ReactOS, SkyOS e outros. AbiWord foi criado inicialmente pela SourceGear Corporation com o componente AbiSuite. SourceGear se graduou a interesses financeiro e deixou o projeto do AbiWord para uma equipe de voluntários.

Interface

A interface do AbiWord lembra a interface do Word pré­2007.

O fato é que muitos usuários ainda se sentem mais confortáveis nessa interface do que na nova ";ribbon"; da ferramenta Microsoft.

O AbiWord oferece todos os recursos necessários para quem tem necessidades simples de edição de textos.
