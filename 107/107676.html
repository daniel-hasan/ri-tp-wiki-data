O Memory Stick é um tipo de cartão de memória flash, para armazenamento de imagens e videos de câmeras digitais e de câmeras de video digitais da Sony. As primeiras versões vinham com 4 ou 8 megabytes. Depois, foram criadas outras maiores, de 16, 32, 64, 128 e 256 megabytes. A partir desse tamanho, são os Memory Stick PRO, mais rápidos e seguros, com 256, 512, 1024 (1 GigaByte), 2048 (2 GigaByte) e 4096 (4 GigaByte) megabytes, hoje em dia já é possível encontrar Memory Sticks com memória até 32GB. Tornou­se um dos melhores cartões de memória com melhor custo/benefício do mercado (ao lado do Cartão SD).

A Sony lançou uma versão compacta, compatível com a comum, chamada de Memory Stick Duo.

Memory Stick Micro (M2)

right A Sony e a Sandisk anunciaram em setembro de 2005[1] e lançaram em fevereiro de 2006[2] um novo tipo de cartão de memória, dessa vez voltado para o mercado de celulares. O Memory Stick Micro tem 25% do tamanho de um Memory Stick Duo. Com o uso de um adaptador, é possível utilizar o Memory Stick Micro em outros dispositivos compatíveis com o formato original.
