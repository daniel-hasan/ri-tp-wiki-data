WGS 84

WGS 84 (World Geodetic System) é uma norma usada em cartografia de origem geocêntrica utilizado pelo GNSS do DoD, e pelo Sistema de Posicionamento Global ­ (GPS), definida em 1984. É composta por um sistema de coordenadas para a Terra, uma superfície de referência esferoidal padrão (a base ou elipsóide de referência) para dados de altitude, e uma superfície gravitacional equipotencial (o geoide) que define o nível médio do mar.
