O projeto de arquitetura é o processo pelo qual uma obra de arquitetura é concebida e também a sua representação final. É considerada a parte escrita de um projeto.

Etapas de um projeto arquitetônico

Um projeto arquitetônico consiste em 5 etapas:

  1. Levantamento de dados
  2. Estudo preliminar
  3. Anteprojeto
  4. Projeto Legal
  5. Projeto Executivo

Composição

O projeto arquitetônico é essencial para se fazer qualquer construção. É constituído de:

  • Implantação
  • Plantas de cobertura
  • Planta baixa
  • Cortes e elevações superiores
  • Layout gráfico
