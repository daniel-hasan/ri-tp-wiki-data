Dr. Mario 64 é um jogo para Nintendo 64 lançado no primeiro semestre de 2001. O jogo é uma mistura do tetris clássico com outros elementos consagrados do gênero puzzle.

Dr. Mario 64 é uma versão mais recente do Dr. Mario para NES (que também foi lançado para Game Boy Color e SNES), nessa nova versão o som e os gráficos são as melhorias mais significativas, embora o sistema todo tenha sido melhorado.

Os blocos do tetris foram substituídos por pequenas criaturas que representam os vírus e o Dr. Mario, como o título de ";Dr."; sugere, deve combatê­los. No jogo, além do Mario o único personagem bastante conhecido é o Wario, o resto são uma maioria de personagens até então desconhecidos do público.

O mais legal do jogo é a opção multiplayer que suporta até quatro jogadores simultaneamente, mas além dessa ainda existem mais seis modos de jogo:

  • Classic: esse é o modo tradicional, o jogador vai jogando e jogando até cansar.
  • Story: esse é o modo que conta a história do jogo. O enredo é diferente dependendo do personagem que você escolher para desvendar a trama, estarão disponíveis para isso o Dr. Mario e o Wario.
  • Flash: nesse modo de jogo o objetivo é destruir apenas as criaturas que estiverem piscando.
  • Vs. Computer: nesse modo você joga contra um personagem controlado pelo videogame.
  • Score Attack: nesse modo de jogo o objetivo é fazer o máximo de pontos no menor espaço de tempo.
  • Marathon: nesse modo a pilha de baixo vai subindo a medida que mais peças vão caindo, o teste aqui é ver até onde você agüenta sem perder o jogo.
