Kalango Linux é uma distribuição de Linux brasileira baseada no Kubuntu, desenvolvida por colaboradores. O Kalango Linux é um Live CD, ou seja, o sistema roda inteiramente do CD sem necessidade de instalação, apesar disso pode perfeitamente ser instalado no HD, como qualquer distribuição tradicional.

O sistema conta com um grande acervo de softwares para atender as necessidades dos desktops de qualquer meio, seja ele residencial, corporativo ou governamental.

Ao realizar um boot com o CD, ele reconhece automaticamente todo o hardware do computador e configura automaticamente, não importa qual a configuração, se os componentes forem compatíveis com o Linux o sistema o reconhecerá e se encarregará de deixa­lo funcionando sem exatamente nenhuma intervenção do usuário. Além disso caso esteja conectado em uma rede com um servidor DHCP, após o carregamento do sistema já poderá navegar na internet sem nenhuma configuração adicional.

O foco é sempre a facilidade de uso, para isso foi simplificado ao máximo a distribuição de modo que um usuário inexperiente possa realizar tarefas do dia­a­dia como: navegar na internet, receber e­mail, bater papo, escutar músicas, ver vídeos e muito mais. Os programas são fáceis de achar devido a organização e padronização dos menus.

O Kalango conta hoje com os melhores softwares livres disponíveis nas mais diversas áreas, como amaroK (Player multimídia), K3B (Gravação de CD e DVD), Azureus (Cliente para bittorrent), OpenOffice (Suíte de escritório), Gimp (Tratamento de imagens), Inkscape (Desenho vetorial), Firefox (Navegador web), Thunderbird (Cliente de email) e muito mais. Também conta com o SGP (Simples Gerenciador de Pacotes), na qual você pode instalar vários programas com poucos cliques.

O ambiente padrão é o KDE (K Desktop Environment), que é um completo desktop, acompanha um painel de controle com muitas opções de personalização e configuração, apelo visual e um conjunto de programas que juntos formam uma solução completa para os desktops em geral. Acompanham também os ambientes gráficos Gnome, IceWM e WindowMaker.

Características da atual versão

Após 1 ano de desenvolvimento a versão 3.3 do Kalango Linux ficou pronta. Essa versão conta com BrOffice.org 2.0.4, KDE 3.5.5, Linux 2.6.17.8­slh­up­1, dentre outras novidades.

O foco da equipe criadora permaneceu na estabilidade, mas dessa vez um pouco mais voltado para segurança.

O Kurumin NG

Leandro Soares dos Santos abandonou o projeto Kalango e se juntou à equipe do Kurumin NG, uma nova geração do Kurumin, que é baseada no Kubuntu 8.04.

Depois de alguns meses de desenvolvimento, este novo projeto foi também abandonado em meio a críticas ligação externa.
