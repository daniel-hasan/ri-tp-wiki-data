Yahoo! Messenger (Y!M ou YIM) é um programa de software através do qual pessoas podem conversar, se escrever e se ver à distância. O software é da empresa Yahoo!, e também se integra à rede do Windows Live Messenger. Está na versão 11 e possui recursos como integração com o Flickr e player de mídia, que exibe imagens e reproduz vídeos de sites como o YouTube quando um link é enviado a outro usuário.

Existe também o Yahoo! Messenger para a Web que é um messenger que não precisa ser instalado, já que funciona diretamente no navegador, parecido com o Windows Live Messenger para a web.

  • Windows Live Messenger, principal concorrente do Yahoo! Messenger, que também se integra ao próprio.
