Uso do Super Game Boy 2

Super Game Boy é um acessório para Super NES que permite ao jogador utilizar qualquer pacote de jogo (cartucho) de Game Boy e alguns do Game Boy Color no Super NES, e jogá­lo na tela da televisão em cores. Alguns jogos com o logotipo do Super Game Boy tem funções exclusivas, mas os jogos de Game Boy sem o logotipo também funcionam. O Super Game Boy também tem funções exclusivas paras todos os jogos. foi lançada uma segunda versão intitulada Super Game Boy 2, onde o hardware foi melhorado para atender mais demandas de jogos do Game Boy Color.
