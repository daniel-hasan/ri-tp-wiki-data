Em linguística, modo verbal é uma função gramatical (e, especificamente, morfológica) de verbos, usado para sinalizar uma modalidade.[1][2][3] Ou seja, é o uso de inflexões verbais que permitem expressar uma atitude para o que se quer dizer (por exemplo, se pretende­se declarar algo como fato, desejo, comando etc). Menos comumente, o termo é usado de forma mais ampla para permitir a expressão sintática da modalidade ­ ou seja, o uso de frases não flexionadas.
