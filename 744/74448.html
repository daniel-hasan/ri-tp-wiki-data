Cloreto em química inorgânica é uma espécie iónica formada por um átomo de cloro carregado negativamente, com estado de oxidação ­1. Por extenção é um composto iónico formado por este anião ou com uma estrutura formalmente similar, ou seja, com ligação covalente entre um átomo de cloro e um elemento menos electronegativo.

A nomenclatura oficial para os compostos que apresentam este anião é:

Cloreto de ( nome do elemento ligado ao cloro)

Exemplos:

CaCl2 : Cloreto de cálcio

HCl : Cloreto de hidrogênio

Compostos

  • Ácido Clorídico ( HCl )
  • cloreto de lítio ( LiCl )
  • cloreto de sódio ( NaCl )
  • cloreto de potássio ( KCl )
  • cloreto de prata (AgCl)
  • cloreto de cálcio ( CaCl2 )
  • cloreto de magnésio ( MgCl2 )
  • cloreto de estrôncio ( SrCl2 )
  • cloreto de bário ( BaCl2 )
  • cloreto de rádio ( RaCl2 )
