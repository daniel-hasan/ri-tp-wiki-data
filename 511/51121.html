Um cliente ';';';VNC';';'; sendo executado no [[KDE]]. Virtual Network Computing (ou somente VNC) é um protocolo desenhado para possibilitar interfaces gráficas remotas. Através deste protocolo um usuário pode conectar­se a um computador remotamente, e utilizar as suas funcionalidades visuais como se estivesse sentado em frente do computador.

Algumas das aplicações práticas incluem a ';assistência remota'; ao usuário remoto.

Uma das grandes vantagens é poder fazer a conexão de diferentes ambientes UNIX (Linux e outros) em WinNT (Windows X).
