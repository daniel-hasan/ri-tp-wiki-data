VisualBoyAdvance (VBA) é um Emulador de Game Boy Advance Game Boy, Super Game Boy e Game Boy Color. É um Software Livre. Atualmente é o emulador mais popular de GBA, contendo diversas funções, desde a gravação de demos e sons até o uso de filters gráficos para deixar o visual dos jogos ainda melhor. Para um bom desempenho do emulador, no caso de executar ROMs de Game Boy Advance é recomendado um processador superior a 500 Mhz.
