A Geography Markup Language (GML) (em português, linguagem de marcação de geografia), utiliza XML para exprimir características geográficas. Pode servir de linguagem de modelação para sistemas geográficos e como um formato aberto para troca de informação geográfica.

A GML é definida pelo Consórcio Geoespacial Livre.
