O Quarenta e um (41) é o número natural que segue o 40 e precede o 42.

Propriedades matemáticas:

  • É o 13º número primo, depois do 37 e antes do 43.
  • É o 7º número primo de Sophie Germain
  • É o 3º número primo de Newman­Shanks­Williams
  • Pelo polinómio f(n) = n2 + n + 41 obtêm­se primos quando n está entre 0 e 41.
  • É a soma de dois quadrados, 41 = 4² + 5².
  • É a soma dos primeiros seis primos, e a soma de três primos consecutivos, 41 = 11 + 13 + 17.

Características:

  • É o número atómico do nióbio (Nb).
