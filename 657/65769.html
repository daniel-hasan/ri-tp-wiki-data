Tela de execução do ClamTk, uma interface gráfica para o ClamAV ClamWin, uma versão para Windows do ClamAV O Clam Antivirus é um antivírus open source que possui versões para Windows (o ClamWin) e Linux, com atualizações diárias. É disponibilizado pela licença GPL. O Clamav para Linux, pode ser usado pela linha de comando, ou baixando um pacote especial para ser usada uma interface gráfica, como na imagem ao lado, pelo pacote ClamTK.
