InterBase é um sistema gerenciador de banco de dados relacionais da Borland. O Interbase é uma alternativa aos bancos de dados tradicionais como Access ou SQL Server. Ele pode ser instalado em sistemas operacionais Microsoft Windows, Linux, Mac OS X e Sun Solaris. Além de não ser pesado é relativamente rápido e suporta bancos de dados de grande tamanho (maiores que 2 Gigabytes).

Em 2000 a Borland liberou o código da versão 6.0, mas as posteriores voltaram a ter licença proprietária. Desta versão 6.0 foi criado o Banco de Dados Open source Firebird. O InterBase em sua primeira versão foi criado com o propósito de ser um banco de dados acadêmico, facilitando a operação e aprendizado por parte dos interessados. Hoje em dia é muito utilizado também no ambiente corporativo.
