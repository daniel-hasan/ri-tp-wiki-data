A base de um sistema de numeração é uma certa quantidade de unidades que deve constituir uma unidade de ordem imediatamente superior. Um sistema de numeração é um conjunto de princípios constituindo o artifício lógico de classificação em grupos e subgrupos das unidades que formam os números. Os sistemas de numeração tem seu nome derivado da sua base, ou seja, o sistema binário tem base dois, o sistema septimal tem base sete e o decimal tem base dez.[1]
