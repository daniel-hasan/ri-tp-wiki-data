Adium é um mensageiro instantâneo para Mac OS X com suporte a múltiplos protocolos através da biblioteca libpurple. É escrito utilizando a API do Cocoa, e está liberada sob a Licença GPL.

Protocolos

Adium suporta diversos protocolos de mensageiros instantâneos, entre eles:

  • AOL Instant Messenger (via protocolos OSCAR ou TOC)
  • Bonjour (Antes Rendezvous)
  • Gadu­Gadu
  • Google Talk
  • ICQ (via OSCAR)
  • XMPP
  • Lotus Sametime
  • .Mac
  • .NET Messenger Service (MSN Messenger)
  • Novell
  • Yahoo! Japan
  • Yahoo! Messenger
  • Zephyr
